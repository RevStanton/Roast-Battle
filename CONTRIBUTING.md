# Contributing to AI Roast Battle Arena

Thank you for your interest in contributing! This project is all about making people laugh, so let's keep it fun and collaborative.

## How to Contribute

### Adding New Personas

1. Edit the `personas` array in `index.html`
2. Follow this structure:
```javascript
{
    id: 'unique-id-here',
    emoji: 'ðŸŽ­', // Pick a fun emoji
    name: 'Persona Name',
    description: 'Short, punchy description for the UI',
    style: 'Detailed personality traits, speaking style, and quirks for the AI to embody'
}
```

3. Make sure your persona:
   - Has a distinct personality
   - Isn't based on a real person (fictional archetypes only!)
   - Would be funny in a roast battle
   - Has a clear speaking style

### Improving the UI

- Keep the bold, energetic design aesthetic
- Test on mobile and desktop
- Ensure accessibility (contrast, keyboard navigation)
- Add comments for complex CSS

### Bug Fixes

1. Fork the repository
2. Create a branch: `git checkout -b fix-something`
3. Make your changes
4. Test thoroughly
5. Submit a pull request with a clear description

### Feature Ideas

Before building a major feature, open an issue to discuss it! Some ideas we'd love to see:
- User voting system
- Shareable battle cards
- Sound effects
- More battle rounds options
- Custom persona creator

## Code Style

- Keep it simple - this is a single-file app
- Comment your code
- Use meaningful variable names
- Test in multiple browsers

## Testing

Before submitting:
1. Test the full battle flow
2. Try different persona combinations
3. Check mobile responsiveness
4. Verify all buttons work
5. Make sure error handling works

## Questions?

Open an issue or reach out! We're here to have fun and build something cool together.

## Code of Conduct

- Be kind and respectful
- Keep humor appropriate (funny, not offensive)
- Help others learn
- Credit others' ideas
- Have fun!

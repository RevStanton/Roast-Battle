## ğŸ”§ How It Works

The app uses the Anthropic Claude API to generate roasts in real-time. Each persona has:
- A unique personality and speaking style
- Context awareness (responds to previous roasts)
- Topic-specific roasting ability

**Architecture**:
- **Frontend**: Single HTML page with vanilla JavaScript
- **Backend**: Vercel serverless function (`api/generate-roast.js`)
- **API**: Anthropic Claude Sonnet 4 for roast generation

Your API key is stored securely in Vercel environment variables and never exposed to the browser.
